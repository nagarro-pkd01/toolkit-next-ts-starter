# Cursor Skills Guide

## When to Use Skills

- Use skills for repeatable multi-step tasks
- Keep each skill focused on one job-to-be-done
- Store operational assumptions and validation steps

## Suggested Skill Catalog

- `release-readiness`: verify CI, hooks, changelog, and publish prerequisites
- `multi-client-bootstrap`: scaffold new client config and brand tokens
- `incident-triage`: collect logs, isolate blast radius, and suggest rollback path

## Maintenance

- Version skill prompts with changelog notes
- Review skills quarterly for stale commands or tools
- Include safety constraints for destructive operations
