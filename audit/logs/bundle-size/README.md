# Bundle size logs

Generated by `scripts/reportBundleSize.mjs` (via `npm run analyze:bundle` or the post-commit hook).

| File | Purpose |
|------|---------|
| `latest.json` | Snapshot used to compute the next diff (gitignored) |
| `history.jsonl` | One JSON record per run (gitignored) |
| `report.log` | Human-readable append-only log (gitignored) |

Skip locally: `SKIP_BUNDLE_SIZE=1 git commit ...`

See `docs/BUNDLE_SIZE.md`.
